IDIOM: A collection of words

Note that this list is a recent work in progress, and suggests intervention in the sense that I would like to both expand on this list, while at the same time embed the information in memory. So I like to think of this sort of like a set of flash-cards, but I’ll be adding more mathematicians, philosophers, sciences and etc. Again, the list started with a word I heard on a radio or television show or perhaps a website, and in stead of collecting lists of books, I thought that it would be a good idea to collect names that can be looked up via various media formats such as Wikipedia, an encyclopedia, Youtube and so on.

I like the idea of going through my book collection for word sources. The idea is to focus on source materials for common and uncommon idiom.
